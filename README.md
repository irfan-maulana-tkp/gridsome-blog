## Development

We are using [pnpm](https://pnpm.js.org/), please read the [Installation](https://pnpm.js.org/en/installation) section first.

``` bash
# install dependencies
pnpm install

# serve with demo page
pnpm run develop

# build for production
pnpm run build
```
